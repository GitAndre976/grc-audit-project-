# Audit Checklist – CloudCRM

| Control ID | Audit Question | Evidence Required |
|------------|--------------|-------------------|
| C1 | Is customer data encrypted at rest? | Database configuration / screenshots |
| C2 | Are strong password policies enforced? | Password policy settings |
| C3 | Is Multi-Factor Authentication enabled? | MFA configuration |
| C4 | Are S3 buckets properly restricted? | Access policy configuration |
| C5 | Is logging and monitoring enabled? | Log samples / monitoring dashboard |

## Methodology
Audit questions are designed to verify the implementation of controls based on ISO/IEC 27001.
