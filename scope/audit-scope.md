# Audit Scope – CloudCRM

## Framework Reference
This audit is based on ISO/IEC 27001.

## 1. Objective
The objective of this audit is to evaluate the effectiveness of security controls in the CloudCRM SaaS platform, focusing on data protection, access control, and monitoring.

## 2. Scope
- Web application
- Authentication
- Database
- Out of scope: Cloud infrastructure and Cloud OS

## 3. Key Assets
- Customer data
- User credentials

## 4. Main Risks
- Unauthorized access
- Data breach
